Chapter 5: Basics of JavaScript Functions

Introduction

Functions are one of the fundamental building blocks in JavaScript. A function is a reusable block of code designed to perform a particular task. Functions can take inputs, perform operations, and return outputs. This chapter will cover the basics of defining and using functions in JavaScript.

Defining a Function

In JavaScript, a function can be defined using the function keyword followed by a name, parentheses, and a block of code.

// Example of a function definition
function greet() {
    console.log("Hello, World!");
}

Calling a Function

To execute a function, you need to call it by its name followed by parentheses.

// Example of calling a function
greet(); // Output: Hello, World!

Function Parameters

Functions can accept parameters, which are values passed to the function when it is called. These parameters can be used within the function.

// Example of a function with parameters
function greetUser(name) {
    console.log("Hello, " + name + "!");
}

// Calling the function with an argument
greetUser("Alice"); // Output: Hello, Alice!

Returning Values

Functions can return values using the return statement. Once a function returns a value, it exits immediately.

// Example of a function that returns a value
function add(a, b) {
    return a + b;
}

// Calling the function and storing the returned value
let result = add(5, 3);
console.log(result); // Output: 8

Function Expressions

In JavaScript, functions can also be defined using function expressions. A function expression is when a function is assigned to a variable.

// Example of a function expression
const square = function(x) {
    return x * x;
};

// Calling the function
console.log(square(4)); // Output: 16

Anonymous Functions

An anonymous function is a function without a name. Function expressions often use anonymous functions.

// Example of an anonymous function
const multiply = function(a, b) {
    return a * b;
};

console.log(multiply(2, 3)); // Output: 6

Function Scope

Variables declared within a function are local to that function and cannot be accessed outside of it. This is known as function scope.

// Example of function scope
function exampleScope() {
    let localVariable = "I'm local!";
    console.log(localVariable); // Output: I'm local!
}

exampleScope();
console.log(localVariable); // ReferenceError: localVariable is not defined
