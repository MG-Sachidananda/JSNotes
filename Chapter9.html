Chapter 9: JavaScript Numbers, Number Properties, and Number Methods

Introduction to Numbers

In JavaScript, numbers are a data type that includes both integer and floating-point values. The number type also supports special values like NaN (Not a Number), Infinity, and -Infinity.

Number Properties

Numbers have properties that provide information about their characteristics.

  • Number.MAX_VALUE: The largest possible value a number in JavaScript can have: approximately 1.79E+308.
  • Number.MIN_VALUE: The smallest possible value a number in JavaScript can have: approximately 5e-324.
  • Number.POSITIVE_INFINITY: Represents infinity.
  • Number.NEGATIVE_INFINITY: Represents negative infinity.
  • Number.NaN: Represents "Not-a-Number".
  • Number.EPSILON: The difference between 1 and the smallest value greater than 1 that is representable.
  • Number.MIN_SAFE_INTEGER: The minimum safe integer in JavaScript: -(2^53 - 1).
  • Number.MAX_SAFE_INTEGER: The maximum safe integer in JavaScript: 2^53 - 1.

Number Methods

JavaScript provides various built-in methods to work with numbers.

1. Conversion Methods

Number(value): Converts a value to a number.

parseFloat(string): Parses a string and returns a floating-point number.

parseInt(string, radix): Parses a string and returns an integer of the specified radix (base).

// Example of using conversion methods
let num1 = Number("123");
console.log(num1); // Output: 123

let num2 = parseFloat("123.45");
console.log(num2); // Output: 123.45

let num3 = parseInt("101", 2);
console.log(num3); // Output: 5

2. Checking and Comparing Methods

Number.isNaN(value): Determines whether the passed value is NaN.

Number.isFinite(value): Determines whether the passed value is a finite number.

Number.isInteger(value): Determines whether the passed value is an integer.

Number.isSafeInteger(value): Determines whether the passed value is a safe integer.

// Example of using checking and comparing methods
console.log(Number.isNaN(NaN)); // Output: true
console.log(Number.isFinite(123)); // Output: true
console.log(Number.isInteger(123.45)); // Output: false
console.log(Number.isSafeInteger(9007199254740991)); // Output: true

3. Formatting Methods

toFixed(digits): Formats a number using fixed-point notation.

toExponential(digits): Formats a number in exponential notation.

toPrecision(precision): Formats a number to a specified precision.

// Example of using formatting methods
let num = 123.456;

console.log(num.toFixed(2)); // Output: 123.46
console.log(num.toExponential(2)); // Output: 1.23e+2
console.log(num.toPrecision(4)); // Output: 123.5

4. Miscellaneous Methods

valueOf(): Returns the primitive value of a Number object.

// Example of using valueOf method
let numObj = new Number(123);
console.log(numObj.valueOf()); // Output: 123

BigInt

BigInt is a built-in object that provides a way to represent whole numbers larger than Number.MAX_SAFE_INTEGER. BigInts are created by appending n to the end of an integer or by calling the BigInt() function.

1. Creating BigInt

BigInt(value): Converts a value to a BigInt.

// Example of creating BigInt
let bigInt1 = BigInt(12345678901234567890);
let bigInt2 = 12345678901234567890n;
console.log(bigInt1); // Output: 12345678901234567890n
console.log(bigInt2); // Output: 12345678901234567890n

2. BigInt Methods

BigInt inherits methods from the Number object.

  • toLocaleString(locale, options): Returns a string with a language-sensitive representation of this number.
  • toString(radix): Returns a string representing the specified BigInt object in the specified radix (base).
  • valueOf(): Returns the primitive value of the BigInt object.
// Example of using BigInt methods
let bigInt = 12345678901234567890n;

console.log(bigInt.toString(16)); // Output: ab54a98ceb1f0ad2
console.log(bigInt.valueOf()); // Output: 12345678901234567890n
