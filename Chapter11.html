Chapter 11: JavaScript Booleans and Comparisons

Introduction to Booleans

In JavaScript, a boolean represents one of two values: true or false. Booleans are often used in conditional testing.

Boolean Values

Boolean values are typically the result of comparisons and logical operations.

let isTrue = true;
let isFalse = false;

console.log(isTrue); // Output: true
console.log(isFalse); // Output: false

Boolean Conversion

JavaScript can convert values of other types to boolean values using the Boolean() function or double negation !!.

console.log(Boolean(0)); // Output: false
console.log(Boolean(1)); // Output: true
console.log(Boolean("")); // Output: false
console.log(Boolean("hello")); // Output: true
console.log(!!0); // Output: false
console.log(!!1); // Output: true

Truthy and Falsy Values

In JavaScript, certain values are considered "truthy" or "falsy" when evaluated in a boolean context.

Falsy Values

The following values are considered falsy:

  • false
  • 0 (zero)
  • "" or '' (empty string)
  • null
  • undefined
  • NaN (Not-a-Number)
console.log(Boolean(false)); // Output: false
console.log(Boolean(0)); // Output: false
console.log(Boolean("")); // Output: false
console.log(Boolean(null)); // Output: false
console.log(Boolean(undefined)); // Output: false
console.log(Boolean(NaN)); // Output: false

Truthy Values

All values that are not falsy are considered truthy. Some examples include:

  • true
  • Non-zero numbers (e.g., 42)
  • Non-empty strings (e.g., "hello")
  • Objects (e.g., {})
  • Arrays (e.g., [])
console.log(Boolean(true)); // Output: true
console.log(Boolean(42)); // Output: true
console.log(Boolean("hello")); // Output: true
console.log(Boolean({})); // Output: true
console.log(Boolean([])); // Output: true

Comparison Operators

Comparison operators are used to compare two values. They return a boolean value.

  • == (equal to)
  • === (strict equal to)
  • != (not equal to)
  • !== (strict not equal to)
  • > (greater than)
  • >= (greater than or equal to)
  • < (less than)
  • <= (less than or equal to)
let a = 5;
let b = 10;

console.log(a == b); // Output: false
console.log(a === b); // Output: false
console.log(a != b); // Output: true
console.log(a !== b); // Output: true
console.log(a > b); // Output: false
console.log(a >= b); // Output: false
console.log(a < b); // Output: true
console.log(a <= b); // Output: true

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • && (logical AND)
  • || (logical OR)
  • ! (logical NOT)
let x = true;
let y = false;

console.log(x && y); // Output: false
console.log(x || y); // Output: true
console.log(!x); // Output: false

Conditional (Ternary) Operator

The conditional operator (?) assigns a value to a variable based on a condition.

let age = 18;
let canVote = (age >= 18) ? "Yes" : "No";
console.log(canVote); // Output: Yes

Practical Examples

Booleans and comparisons are fundamental in controlling the flow of a program.

let age = 20;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

let isMember = true;
let discount = isMember ? 10 : 0;
console.log("Discount: " + discount + "%");
