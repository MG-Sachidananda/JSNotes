Chapter 3: JavaScript Data Types

Introduction

In JavaScript, data types are categorized into two main types: primitive and object types. Understanding these data types is essential for effective programming.

Primitive Data Types

Primitive data types are the most basic data types in JavaScript. They include:

  • String: Represents textual data. Strings are created using single quotes (' '), double quotes (" "), or backticks (` `).
  • Number: Represents both integer and floating-point numbers.
  • Boolean: Represents logical entities and can have two values: true or false.
  • Null: Represents the intentional absence of any object value. It is a special keyword and evaluates to null.
  • Undefined: Indicates that a variable has not been assigned a value. Variables are initialized with undefined by default.
  • Symbol: Represents a unique and immutable value, primarily used for object property keys.
  • BigInt: Represents whole numbers larger than 2^53 - 1 (the largest number JavaScript can safely represent with the Number primitive).

Examples of Primitive Data Types

// String
let name = "Alice";
let greeting = `Hello, ${name}!`;

// Number
let age = 30;
let temperature = 98.6;

// Boolean
let isActive = true;
let hasLicense = false;

// Null
let emptyValue = null;

// Undefined
let uninitialized;

// Symbol
let sym = Symbol('description');

// BigInt
let bigIntValue = 1234567890123456789012345678901234567890n;

Object Types

Object types are more complex and can store collections of data and more complex entities. The main types of objects in JavaScript include:

  • Object: A collection of properties, where each property is a key-value pair.
  • Array: An ordered list of values, indexed by numbers.
  • Function: A block of code designed to perform a particular task. Functions themselves are objects in JavaScript.
  • Date: Represents dates and times.
  • RegExp: Represents regular expressions, used for pattern matching in strings.
  • Map: A collection of keyed data items, where keys can be of any type.
  • Set: A collection of unique values, where each value may occur only once.

Examples of Object Types

// Object
let person = {
    name: "Alice",
    age: 30,
    isActive: true
};

// Array
let numbers = [1, 2, 3, 4, 5];

// Function
function greet(name) {
    return `Hello, ${name}!`;
}

// Date
let now = new Date();

// RegExp
let pattern = /ab+c/;

// Map
let map = new Map();
map.set('name', 'Alice');
map.set('age', 30);

// Set
let set = new Set();
set.add(1);
set.add(2);
set.add(3);
