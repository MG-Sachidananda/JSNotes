Chapter 1: Basics of JavaScript

What is JavaScript?

JavaScript is a versatile and widely-used programming language primarily used for web development. It allows developers to create dynamic and interactive web pages. Originally developed by Netscape as a means to add dynamic and interactive elements to websites, JavaScript has evolved to become one of the three core technologies of web development, alongside HTML and CSS.

JavaScript is executed on the client side by the web browser, which means it can manipulate the content and behavior of web pages directly without requiring communication with the server. This makes it an essential tool for creating responsive and engaging user experiences.

Some key features of JavaScript include:

  • Lightweight and Interpreted: JavaScript is an interpreted language, meaning code can be executed without a prior compilation step. This makes it flexible and easy to work with.
  • Event-Driven: JavaScript can respond to user actions like clicks, mouse movements, and keyboard inputs, making web pages interactive.
  • Prototype-Based: JavaScript uses prototypes for inheritance, allowing objects to share properties and methods.
  • First-Class Functions: Functions in JavaScript are treated as first-class objects, meaning they can be assigned to variables, passed as arguments, and returned from other functions.

What is a Document?

In the context of web development, a document refers to the HTML document loaded into the web browser. This document forms the structure and content of the web page. The Document Object Model (DOM) represents this document as a hierarchical tree structure, where each element in the HTML is a node in the tree.

The DOM is an essential concept for JavaScript because it provides a way to access and manipulate the content, structure, and styles of web pages programmatically. Using JavaScript, developers can modify the DOM to change how a web page looks and behaves.

Some common operations that JavaScript can perform on the DOM include:

  • Selecting Elements: JavaScript can select HTML elements using methods like document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), and document.querySelector().
  • Modifying Content: JavaScript can change the content of HTML elements using properties like innerHTML and textContent.
  • Changing Styles: JavaScript can alter the styles of HTML elements using the style property.
  • Handling Events: JavaScript can respond to user interactions by attaching event listeners to HTML elements using methods like addEventListener().

Here's a simple example demonstrating some DOM manipulation:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>DOM Manipulation Example</title>
</head>
<body>
    <h1 id="title">Hello, World!</h1>
    <button id="changeTitleButton">Change Title</button>

    <script>
        // Selecting the elements
        const title = document.getElementById('title');
        const button = document.getElementById('changeTitleButton');

        // Adding an event listener to the button
        button.addEventListener('click', function() {
            // Changing the content and style of the title
            title.textContent = 'Title Changed!';
            title.style.color = 'red';
        });
    </script>
</body>
</html>

In this example, when the button is clicked, the text content and color of the heading are changed using JavaScript.
