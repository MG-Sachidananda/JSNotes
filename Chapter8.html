Chapter 8: JavaScript Strings, Template Strings, Properties, and Methods

Introduction to Strings

In JavaScript, a string is a sequence of characters enclosed within single quotes '' or double quotes "". Strings are immutable, meaning their content cannot be changed once created.

String Properties

Strings have properties that allow you to access information about the string.

  • length: Returns the length of the string.
// Example of using string property
let message = "Hello, World!";
console.log(message.length); // Output: 13

String Methods

JavaScript provides many built-in methods to manipulate strings.

1. Concatenation

concat(): Combines two or more strings and returns a new string.

// Example of using concat method
let str1 = "Hello";
let str2 = "World";
let greeting = str1.concat(", ", str2);
console.log(greeting); // Output: Hello, World

2. Accessing Characters

charAt(index): Returns the character at the specified index.

// Example of using charAt method
let str = "JavaScript";
console.log(str.charAt(0)); // Output: J

3. Substrings

substring(startIndex, endIndex): Returns the substring between startIndex and endIndex (endIndex not included).

// Example of using substring method
let str = "JavaScript";
console.log(str.substring(0, 4)); // Output: Java

4. Searching

indexOf(searchValue): Returns the index of the first occurrence of searchValue within the string.

lastIndexOf(searchValue): Returns the index of the last occurrence of searchValue within the string.

// Example of using indexOf and lastIndexOf methods
let str = "Hello, World!";
console.log(str.indexOf("o")); // Output: 4
console.log(str.lastIndexOf("o")); // Output: 8

5. Case Conversion

toLowerCase(): Converts a string to lowercase.

toUpperCase(): Converts a string to uppercase.

// Example of using toLowerCase and toUpperCase methods
let str = "Hello, World!";
console.log(str.toLowerCase()); // Output: hello, world!
console.log(str.toUpperCase()); // Output: HELLO, WORLD!

6. Trimming

trim(): Removes whitespace from both ends of a string.

trimStart(): Removes whitespace from the start of a string.

trimEnd(): Removes whitespace from the end of a string.

// Example of using trim, trimStart, and trimEnd methods
let str = "   Hello, World!   ";
console.log(str.trim()); // Output: Hello, World!
console.log(str.trimStart()); // Output: "Hello, World!   "
console.log(str.trimEnd()); // Output: "   Hello, World!"

7. Splitting

split(separator): Splits a string into an array of substrings using the specified separator.

// Example of using split method
let str = "Hello, World!";
let words = str.split(", ");
console.log(words); // Output: ["Hello", "World!"]

8. Replacing

replace(searchValue, newValue): Replaces the first occurrence of searchValue with newValue.

replaceAll(searchValue, newValue): Replaces all occurrences of searchValue with newValue.

// Example of using replace and replaceAll methods
let str = "Hello, World!";
console.log(str.replace("World", "JavaScript")); // Output: Hello, JavaScript!
console.log(str.replaceAll("o", "O")); // Output: HellO, WOrld!

9. Padding

padStart(targetLength, padString): Pads the current string with a specified string from the start to reach the target length.

padEnd(targetLength, padString): Pads the current string with a specified string from the end to reach the target length.

// Example of using padStart and padEnd methods
let str = "5";
console.log(str.padStart(3, "0")); // Output: 005
console.log(str.padEnd(3, "0")); // Output: 500

10. Repeating

repeat(count): Returns a new string with a specified number of copies of the original string.

// Example of using repeat method
let str = "Hello";
console.log(str.repeat(3)); // Output: HelloHelloHello

11. Checking String Content

includes(searchValue): Checks if the string contains searchValue.

startsWith(searchValue): Checks if the string starts with searchValue.

endsWith(searchValue): Checks if the string ends with searchValue.

// Example of using includes, startsWith, and endsWith methods
let str = "Hello, World!";
console.log(str.includes("World")); // Output: true
console.log(str.startsWith("Hello")); // Output: true
console.log(str.endsWith("!")); // Output: true

12. Extracting Parts of a String

slice(startIndex, endIndex): Extracts a section of a string and returns it as a new string. Negative indices count back from the end of the string.

// Example of using slice method
let str = "Hello, World!";
console.log(str.slice(7, 12)); // Output: World
console.log(str.slice(-6, -1)); // Output: World
